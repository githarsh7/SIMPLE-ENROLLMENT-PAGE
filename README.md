# 🌐 Web Development Workshop Enrollment Form

A responsive and modern enrollment form built using **HTML** and **CSS**. This project showcases a clean UI design and a structured layout for registering users for a workshop.

---

## 📌 Features

- Clean and modern user interface  
- Fully responsive design (mobile-friendly)  
- Form with required field validation  
- User input fields (Username, Email, Phone)  
- Seat selection dropdown  
- Submit and Reset buttons  
- Font Awesome icons integration  

---

## Technologies Used

- HTML5  
- CSS3  
- Font Awesome
